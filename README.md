## Task Overview

### Task 5: Personal Loan Acceptance Prediction
**Dataset:** Bank Marketing Dataset  
**Objective:** Predict which customers are likely to accept a personal loan offer.  
**Approach:**  
- Explored key features like age, job, and marital status  
- Encoded categorical features  
- Trained a Decision Tree or Logistic Regression classifier  
- Analyzed results to identify high-potential customer groups  
**Results & Insights:**  
- Certain age groups and job types showed higher acceptance rates
